# BasicClasses - Introduction to Java

This project introduces basic Java programming concepts including class structure, static methods, and user input handling.

## OOP Concepts Demonstrated

- **Basic Class Structure**: Simple class definition with methods
- **Static Methods**: Using static methods for utility functions
- **User Input**: Reading and processing user input
- **Console I/O**: Basic input/output operations

## Project Structure

```
BasicClasses/
├── full_credit/
│   └── Hello.java          # Basic hello world with user input
├── bonus/
│   └── Hello.java          # Enhanced version with additional features
└── extreme_bonus/
    └── Hello.java          # Advanced version with extra functionality
```

## How to Run

### Compile and Run
```bash
# Navigate to the desired directory
cd full_credit

# Compile
javac Hello.java

# Run
java Hello
```

## Learning Objectives

- Understanding basic Java class structure
- Working with static methods
- Handling user input with Scanner
- Basic console output formatting
